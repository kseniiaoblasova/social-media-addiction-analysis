# Social Media Addiction EDA

In this project, I perform a basic-level exploritory data analysis of a "Students' Social Media Addiction" dataset taken from Kaggle.

The dataset includes survey data with students across the globe reporting their social media usage habits and how it affects some aspects of their lives.

Some facts about the dataset:

Population: Students aged 16–25 enrolled in high school, undergraduate, or graduate programs.
Geography: Multi‐country coverage (e.g., Bangladesh, India, USA, UK, Canada, Australia, Germany, Brazil, Japan, South Korea).
Timeframe: Data collected via a one‐time online survey administered in Q1 2025.
